# 🧭 Journeys.
**Stop Planning. Start Exploring.**

 **Journeys** is a modern trip-planning dashboard designed to eliminate the friction of travel prep. it leverages Generative AI to craft highly specific, day-by-day itineraries based on user destinations and dates. 

## ✨ Key Features

* **🤖 AI Itinerary Generation:** Instantly creates realistic, paced travel schedules including local cuisine, geographical intel, and daily activities.
* **🔐 Secure Authentication:** Robust JWT-based authentication system with HTTP-only cookies, password hashing, and email verification workflows.
* **🎒 Smart Packing Tracker:** Interactive, persistent packing lists with category sorting and a visual progress bar.
* **📱 Responsive & Animated UI:** A premium, dark-themed glassmorphic interface fully optimized for mobile and desktop, powered by Tailwind CSS and Framer Motion.
* **🗺️ Geographical Context:** Integrated mapping and location coordinate generation for all planned landmarks.

## 🛠️ Tech Stack

**Frontend:**
* React.js (Vite)
* Tailwind CSS (Styling & Responsive Design)
* Framer Motion (Complex layout animations and page transitions)
* Lucide React (Iconography)
* React Router DOM (Routing & Protection)

**Backend:**
* Node.js & Express.js
* MongoDB & Mongoose (Data modeling & persistence)
* JSON Web Tokens (JWT) (Session management)
* Bcrypt.js (Password cryptography)
* Generative AI API (Itinerary intelligence)
